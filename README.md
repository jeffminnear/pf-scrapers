# pf-scrapers

The data scraping utility used by pilot-fish and pf-meteor

## Getting Started
Install the module with: `npm install pf-scrapers`

```javascript
var pf_scrapers = require('pf-scrapers');
pf_scrapers.awesome(); // "awesome"
```
